# Python Interactive Learning Sample

ブラウザ上で Python の基礎を学び、コードの動きをその場で確認できるシングルページのインタラクティブ教材テンプレートです。

## 🚀 特徴
* **環境構築不要**: HTMLファイル1つで完結しているため、ブラウザ（Chrome, Edge, Safari等）があればどこでも動きます。
* **リアルタイム実行**: エディタに入力した内容をその場でシミュレートし、実行結果を表示します。
* **軽量・高速**: 外部ライブラリを使用せず、ピュアな HTML/CSS/JavaScript のみで構築されています。

## 📂 ファイル構成
* `index.html`: 解説テキスト、スタイリング、および擬似実行エンジンが含まれるメインファイル。

## 🛠 使い方
1.  このリポジトリをクローンするか、`index.html` をダウンロードします。
2.  `index.html` をダブルクリックしてブラウザで開きます。
3.  「インタラクティブ・エディタ」内のコードを書き換え、「実行する」ボタンを押して結果を確認します。

## ⚠️ 注意事項（制限事項）
このサンプルは、JavaScript の `eval()` を用いた簡易的なエミュレーションで動作しています。そのため、以下の点に注意してください。
* **標準Pythonとの違い**: 本物の Python インタプリタではないため、複雑なライブラリ（numpyなど）や高度な構文（クラス定義、複雑な内包表記など）には対応していません。
* **セキュリティ**: 入力された文字列をスクリプトとして実行する仕組みのため、ローカル学習用としてのみ使用してください。

## 🌟 今後の拡張案（カスタマイズ）
より本格的な Python 学習環境にアップグレードしたい場合は、以下のライブラリの導入を検討してください。
* [Pyodide](https://pyodide.org/): WebAssembly を使用して、ブラウザ上で本物の Python 環境を動かせます。
* [Brython](https://brython.info/): Python 3 の文法を JavaScript に変換して実行するライブラリです。
